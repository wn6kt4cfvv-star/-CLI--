# Учебный конфигурационный язык (CLI-инструмент)

Инструмент командной строки для трансляции учебного конфигурационного языка
в формат JSON с вычислением констант на этапе трансляции.

## Возможности
- объявление констант
- массивы с вложенностью
- вычисление констант через `@(имя)`
- обработка синтаксических ошибок
- вывод результата в JSON

## Синтаксис языка

### Числа

[1-9][0-9]*


### Имена

[_a-zA-Z]+


### Объявление константы

имя <- значение;


### Использование константы

@(имя)


### Массивы

{ значение . значение . ... }


### Использование

Запуск инструмента из командной строки:

```bash
python src/config_tool.py --input examples/network.cfg --output out.json

Пример
Сетевая конфигурация

Входной файл (network.cfg):

port <- 8080;
replicas <- { 1 . 2 . 3 };
main_port <- @(port);

Результат трансляции (out.json):

{
  "port": 8080,
  "replicas": [1, 2, 3],
  "main_port": 8080
}
